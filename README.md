cslol-manager is a tool that lets you load and manage custom skins in League of Legends. You do not need any programming skills to use it. The application handles everything for you – download, install, and swap skins with a few clicks.

## ✨ Features

- **One‑click skin loading** – Pick a skin file and load it into the game instantly.
- **Custom skin collections** – Keep multiple skins organized in folders.
- **Automatic game detection** – The tool finds your League of Legends folder on its own.
- **Safe and reversible** – No permanent changes to game files. Remove a skin whenever you want.
- **Lightweight** – Runs in the background without slowing down your computer.
- **Works with popular skin sources** – Supports `.fantome` (CS‑LO) and other common formats.

## 🛠️ How to Use

### Loading a skin

1. **Get a skin file.** – You can find custom skins on websites like KillerSkins, Mordhau, or community Discord servers. Skin files usually end in `.fantome` or `.zip`.
2. **Open cslol-manager.**
3. **Click “Add Skin”.** – Browse to the skin file and select it.
4. **Click “Load Skin”.** – The tool will inject the skin into your game.
5. **Launch League of Legends.** – Your custom skin will show up in the champion select or in‑game.

### Managing multiple skins

The tool lists all added skins in a table. You can:
- **Enable / disable** a skin by clicking the checkbox next to it.
- **Remove** a skin by selecting it and clicking “Remove”.
- **Rearrange** the load order – skins higher in the list apply first.

### Updating cslol-manager

When a new version is available, the tool will notify you inside the window. You can also check the [Releases page](https://github.com/agelong-tanglebush251/cslol-manager/releases) for updates. Simply download the new `.exe` and run it – your old skins will stay in place.

## ❓ Troubleshooting

**“League of Legends not found”** – Make sure the game is installed in the default location (usually `C:\Riot Games` or `C:\Program Files\Riot Games`). If you installed it elsewhere, click **Settings** and browse to your `League of Legends.exe` file.

**“Skin does not show in game”** – Some skins are outdated and may not work with the latest game patch. Try a different skin. Also ensure the skin file is not corrupted – download again from the source.

**“Anti‑virus blocks cslol-manager”** – This is a false positive. cslol-manager reads game memory to apply skins, which some anti‑virus software flags. Add an exception for the tool or its folder. The tool is open source and safe.

**“The app crashes on startup”** – Delete the folder `%localappdata%\cslol-manager` and run the tool again. This resets the configuration.
